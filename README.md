## 🚀 Project Overview
This project was created as part of a collaborative academic task at **Gdańsk University of Technology**. The goal was to implement a functional game loop, character physics, and interactive world elements using the Unity engine.

### Key Features:
* **Smooth Character Controller:** Custom physics-based movement including running, jumping, and collision detection.
* **Level Design:** Interactive environments with platforms, collectibles, and hazards.
* **UI System:** Main menu, HUD, and game-over states.

## ⌨️ Controls
* **WASD** - Movement
* **Space** - Jump
* **Shift** - Dash
* **LMB and C** - Building mode
* **Esc** - Pause Menu
